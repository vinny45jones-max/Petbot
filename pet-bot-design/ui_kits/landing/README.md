# UI Kit — Pet BOT landing page

Модульный одностраничный лендинг для **Pet BOT**. Цель — за 1 скрин объяснить зачем, за 2-3 скролла показать как работает, и привести к кнопке «Открыть в Telegram».

## Состав

| Файл | Что это |
| --- | --- |
| `index.html` | Сборка всех секций в реальный лендинг. Откройте в preview pane. |
| `Header.jsx`  | Sticky хедер с логотипом и CTA |
| `Hero.jsx` | Главный экран: заголовок + sub + фото-коллаж + CTA |
| `HowItWorks.jsx` | 3 шага 01 / 02 / 03 с иконками |
| `Gallery.jsx` | «До → после» — показывает что бот делает с фото |
| `Examples.jsx` | 3 примера готовых объявлений из канала |
| `CTA.jsx` | Финальный блок призыва |
| `Footer.jsx` | Футер с контактами и ссылками |

## Как посмотреть

Откройте `index.html`. Сайт адаптивен от 360px.
