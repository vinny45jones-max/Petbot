# Pet Help — Design System

Дизайн-система и одностраничный лендинг для **Pet Help** — волонтёрского проекта по пристройству кошек и собак в РФ и РБ. Лендинг показывает питомцев, принимает заявки на усыновление и помощь, принимает донаты (прямым переводом на карту) — спецификация в `references/WEBSITE_SPEC.md`.

Параллельно существует **Pet BOT** — отдельный Telegram-инструмент для волонтёров (генерирует объявления о пристройстве по одному фото). Это другой продукт. У сайта и бота **общей БД нет**. Единственная точка интеграции на MVP — Telegram-уведомления о заявках через отдельного бота-нотификатора. Pet BOT упоминается на сайте как сопутствующий инструмент для волонтёров.

Эта папка — не про Pet BOT. Она про **маркетинговый сайт Pet Help**: палитра, типографика, компоненты и готовый UI-кит одностраничного лендинга.

## Проект в одном абзаце

Pet Help — независимое волонтёрское сообщество кураторов, передержек и водителей в РФ и РБ. Они находят кошек и собак на улице или забирают из переполненных приютов, лечат, социализируют и ведут до момента, когда животное попадает домой. Комиссий нет, платёжных провайдеров нет — все сборы идут прямыми переводами на карту конкретного куратора, с открытой отчётностью в Telegram-канале.

Лендинг решает **четыре задачи**:

1. Объяснить, кто такие Pet Help и почему им можно доверять.
2. Показать питомцев, которые ищут дом сейчас, и принять заявку «Хочу забрать».
3. Принять донат (скопировать номер карты в один клик) + показать актуальные нужды.
4. Принять заявку на волонтёрство (передержка / транспорт / вет / выгул).

Все формы улетают в Telegram-чат координаторов через отдельного бота-нотификатора.

## Content fundamentals

### Голос и тон

- **Тёплый, практичный, без драмы.** Не давим на жалость, не используем слова «бедный», «несчастный», «погибает». Пристройство — будничное выполнимое дело, а не трагедия.
- **Уважение и к человеку, и к животному.** Волонтёр — деятельный человек, у которого мало времени. Потенциальный усыновитель — взрослый, которому нужны факты, а не умиление. Животное — живое существо со своим характером, не «товар».
- **Конкретика вместо эпитетов.** «Привит, кастрирован, ходит на поводке» > «очень хороший мальчик».
- **Честность без розовых очков.** В описаниях можно написать «подойдёт в дом без детей», «не любит других кошек» — это уважение к будущему хозяину, а не антиреклама.

### Обращение

- **К пользователю — на «вы» со строчной.** «Заберите домой», «напишите куратору», «ваш город».
- **От проекта — «мы».** «Мы ведём», «мы помогаем», «мы не берём комиссий».
- Название проекта — **Pet Help** (латиницей, через пробел, оба слова с заглавной). Бот — **Pet BOT** (BOT капсом), упоминается на лендинге только в секциях About и Contacts как отдельный инструмент для волонтёров.

### Casing

- Заголовки — русское предложение: заглавная только у первого слова и у имён собственных. **Title Case не используем.**
- Кнопки — с заглавной: «Хочу забрать», «Скопировать номер», «Отправить заявку».
- Номера питомцев/заявок — через `№` без пробела: `№47`.

### Эмодзи

- **В интерфейсе — нет.** Вместо эмодзи используем иконки Lucide (стиль rounded, stroke 1.75–2px).
- Исключение — сервисные пуши в Telegram-чат координаторов (там эмодзи уместны как маркер типа заявки: 🐾 усыновление, ❤️ донат). В самом сайте эмодзи не появляются.

### Словарь

| Используем | Не используем |
| --- | --- |
| питомец, кошка, собака, щенок | зверушка, лапочка, пушистик |
| забрать домой, пристроить, найти семью | отдать, сбагрить, избавиться |
| куратор, волонтёр, передержка | активист, доброволец, хозяин-временник |
| заявка, анкета | форма-заявление, обращение |
| донат, прямой перевод | пожертвование, благотворительный взнос |
| Telegram-канал, Telegram-бот | телега, тг, канал в тг |

### Микро-копирайт: примеры

- Hero headline: **«Каждой лапе — свой дом.»**
- Hero sub: «Мы помогаем кошкам и собакам из приютов и с улиц находить семью. Знакомьтесь с питомцами, забирайте домой или помогайте тем, кто ещё ищет.»
- Primary CTA: **«Посмотреть питомцев»** · Secondary CTA: **«Помочь»**
- В карточке питомца: `Кот · мальчик` / `2 года · средний` / CTA **«Хочу забрать»** (или **«Забронирован»** — disabled).
- В секции доната: «Все сборы — прямые переводы. Мы не используем платёжные провайдеры.»
- Футер: «Сделано волонтёрами. Сайт не юрлицо — все сборы прямые переводы на карту куратора.»

## Visual foundations

### Палитра

Тёплая кремовая база + одна живая акцентная температура + глубокий зелёный нейтрал для текста.

| Роль | Токен | Цвет | Куда |
| --- | --- | --- | --- |
| Фон страницы | `--bg` | `#F7F1E6` (тёплый кремовый) | основной холст |
| Альтернативный фон | `--bg-alt` | `#EDE4D3` | «полосатость» секций |
| Поверхность карточки | `--surface` | `#FFFFFF` | карточки, модалки, формы |
| Sunk-поверхность | `--surface-sunk` | `#F3ECDE` | поля ввода, таб-фон |
| Primary (бренд) | `--brand` | `#E2734A` (терракота-коралл) | CTA, активные табы, акценты |
| Primary hover | `--brand-hover` | `#C85F3B` | hover CTA |
| Brand soft | `--brand-soft` | `#FADFD0` | chip-бейджи высокого приоритета |
| Ink (глубокий нейтрал) | `--ink` | `#1F3B2D` (хвойно-зелёный) | заголовки H1–H3, жирный текст |
| Accent cream | `--accent-cream` | `#F3C78B` (тёплая охра) | иконочные капсулы, декор |
| Accent blush | `--accent-blush` | `#F4C9B6` | подложки у иллюстраций |
| Text primary | `--fg` | `#2A2A28` | body текст |
| Text muted | `--fg-muted` | `#6B6A63` | caption, мета, подписи |
| Border | `--border` | `#E3DACB` | разделители, outline кнопок |
| Success | `--success` | `#3F8F6A` | «пристроен», чек-метки, статусы |

**Правило:** на одном экране одновременно не больше **трёх активных цветов** (кремовая база + ink для текста + терракота для CTA). Остальное — вспомогательное.

### Типографика

- **Display:** **Fraunces** (serif с выразительной модуляцией и засечками). Вес 600 для H1–H2, 500 для H3. Плотный leading (`1.02–1.1`). Используется ТОЛЬКО для заголовков, больших цифр и номеров карт.
- **UI / Body:** **Nunito** (humanist sans, скруглённая). 400 body, 600 акценты, 700 кнопки. `line-height: 1.5`.

> **⚠ Substitution flag.** Брендовой гарнитуры в исходных материалах нет. Fraunces + Nunito подобраны как близкие по характеру к Pawtopia-референсу. Если появится официальный брендинг-кит — заменить в `colors_and_type.css`.

Шкала:

| Токен | Размер | Line-height | Где |
| --- | --- | --- | --- |
| `--fs-display` | 72px | 1.02 | Hero H1 |
| `--fs-h1` | 56px | 1.05 | Section title |
| `--fs-h2` | 40px | 1.1 | Sub-section |
| `--fs-h3` | 28px | 1.15 | Card title (питомцы, истории) |
| `--fs-body-lg` | 20px | 1.5 | Hero sub, важный body |
| `--fs-body` | 17px | 1.55 | Обычный текст |
| `--fs-caption` | 14px | 1.45 | Meta, подписи, tiny |

### Геометрия, радиусы, тени

- **Крупные радиусы.** Карточки — `24px`, кнопки — `14px`, chip — `999px` (капсула), изображения — `20–32px`. Острые углы не используем.
- **Блобы в декоре** — допустимы (мягкое пятно за hero-фото).
- Тени — мягкие, тёплые, в два шага. Синих теней не бывает.
- **Запрещено:** карточки с цветным левым бордером 4px, стеклянные «glass» эффекты повсюду, неоновые градиенты, drop-shadow на иконках.

### Иллюстрация и фотография

- **Только реальные фото животных.** Для лендинга используем фотографии реальных питомцев проекта (в макете — заглушки в `assets/placeholders/`).
- Тёплые жёлто-оранжевые тона предпочтительнее. Холодные фото с синим небом стараемся не ставить — они выбиваются из палитры.
- Декоративные иллюстрации не рисуем сами — только line-art из открытых наборов, если понадобится.

### Движение

- **Ease-out всегда:** `cubic-bezier(0.2, 0.7, 0.2, 1)`.
- Hover: 120ms; появление при скролле: 450ms fade + 8px translate-up; модалка: 220ms fade.
- **Без bounce/spring.** Pet Help — волонтёрский инструмент, не детский сервис.

### Layout

- Desktop: `max-width: 1200px`, поля `32px`.
- Mobile: `16px` поля.
- Вертикальный ритм между секциями: `112–120px` desktop / `64px` mobile.
- Sticky header: прозрачный поверх кремового; при скролле становится `rgba(247, 241, 230, 0.88)` с `backdrop-filter: blur(12px)` и нижним бордером.

## Iconography

- **Набор:** Lucide (stroke 1.75px, rounded linecaps).
- **Размеры:** `16px` inline, `20px` в навигации и табах, `24px` в карточках, `48px` в иконочных капсулах на фоне `--accent-cream` или `--ink`.
- **Цвет:** `currentColor`.
- **Эмодзи как иконки — нет.**
- **Telegram logo** — брендовый синий `#24A1DE`, отдельный SVG.
